ReadMe for DISASM32

This archive contains the Delphi translation of the libdisasm project. For more information on the libdisasm project, please visit:

http://bastard.sourceforge.net/libdisasm.html

Special thanks goes to the team behind the "bastard", for providing such an excelllent product. I would also like to thank "mammon_" for granting me permission to make this conversion available to others.

Required Files:
disasm32.pas
disasm32.res.

A demo project is also included to provide a brief overview of usage.
